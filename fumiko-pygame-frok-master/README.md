# INSTALLATION

You'll need python and pygame for this: http://www.pygame.org/

Windows and Mac users download it here: http://www.pygame.org/download.shtml
Linux users can install it with apt-get oder yum or anything alike.

# RUNNING THE GAME

python start.py
